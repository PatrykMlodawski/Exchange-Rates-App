### Built With

* [Chart.js](https://www.chartjs.org/)

## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```
  
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/PatrykMlodawski/Exchange-Rates-App.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start app
   ```sh
   npm start
   ```
